XFIG Version 3.2 Patchlevel 1
Users Manual (Edition 1.0)

xfig は、大部分の UNIX 互換のプラットホーム上の、 X Window System 上で動作する、対話的な描画ツールです。 これはフリーウェアであり、 anonymous ftp によって入手することができます。 より詳しい説明については、 概要 の章を参照して下さい。

この文書の Operating Manual の章では、 xfig の使用方法などについて解説します。 その他の情報、例えば xfig のインストールの方法などは、 Technical Information の章に記載してあります。

XFIG Version 3.2 Patchlevel 1
Users Manual (Edition 1.0)

xfig is an interactive drawing tool which runs under X Window System Version 11 Release 4 (X11R4) or later, on most UNIX-compatible platforms. It is freeware, and available via anonymous ftp. See section `Introduction' for more details.

In the section `Operating Manual' of this document, we will explain how to use xfig. Other information such as how to install xfig is described in the section `Technical Information'.
